Sobre Mim

Transformar ideias em soluções digitais é o que me motiva a seguir na área de tecnologia. Desde que comecei a estudar desenvolvimento, me interessei não apenas pelo código em si, mas por todo o processo de construção de aplicações que possam facilitar o dia a dia das pessoas.

Como Desenvolvedora Full Stack Jr, venho desenvolvendo conhecimentos tanto no front-end quanto no back-end, buscando entender de forma completa como uma aplicação funciona — desde a interface que o usuário vê até a lógica por trás do sistema.

Atualmente, possuo conhecimentos básicos nas principais tecnologias da área e sigo em constante evolução. Tenho uma rotina de estudos diária, buscando reforçar minha base e melhorar minha forma de resolver problemas.

Acredito que o aprendizado contínuo é essencial na tecnologia, por isso estou sempre em busca de ampliar meus conhecimentos e acompanhar as boas práticas do mercado.

Sou de Sorocaba, gosto de desafios e valorizo organização, dedicação e atenção aos detalhes em tudo o que faço.

Estou em busca de uma oportunidade para colocar meus conhecimentos em prática e crescer profissionalmente na área de desenvolvimento.

Cases de Desenvolvimento

TaskManager - Gerenciador de Tarefas

Aplicação full stack desenvolvida para gerenciamento de tarefas, permitindo criar, visualizar, editar e excluir registros. O projeto foi construído com foco na integração entre front-end e API REST, simulando um sistema real.

  • Front-end: Interface responsiva com HTML, CSS e JavaScript.
  • Back-end: API REST para operações CRUD e comunicação com o banco de dados.
  • Banco de Dados: Armazenamento e gerenciamento das tarefas utilizando SQLite.
